คู่มือสั้นสำหรับติดตั้ง frontend (ไม่เชิงเทคนิค)

เป้าหมาย: ทำให้เครื่องของคุณพร้อมรันหน้า UI ใหม่ (React/Vite) ที่ผมย้ายไว้ที่ `resources/frontend/vaelthorn-ui`.

1) ตรวจสอบว่าเครื่องคุณมี Node.js / npm หรือยัง
- เปิด PowerShell แล้วพิมพ์คำสั่งนี้ แล้วกด Enter:

```
node -v
```
- หากได้หมายเลขเวอร์ชัน (เช่น `v18.x.x`) แปลว่าเครื่องมีแล้ว ข้ามไปขั้นตอนที่ 3.

2) หากยังไม่มี Node.js — ติดตั้งแบบง่าย (Windows)
- วิธีง่าย (มีอินเตอร์เฟซ): ไปที่ https://nodejs.org/ และดาวน์โหลด "LTS" (แนะนำ) แล้วติดตั้งตามขั้นตอนบนหน้าจอ.
- วิธีคำสั่ง (ถ้าคุณใช้ Windows 10/11 และมี `winget`): เปิด PowerShell (Admin ถ้าจำเป็น) แล้วรัน:

```
winget install OpenJS.NodeJS.LTS -e
```

3) ติดตั้งแพ็กเกจโปรเจค (เรียกใช้งานในโฟลเดอร์โปรเจคของคุณ)
- เปิด PowerShell แล้วไปที่โฟลเดอร์โปรเจค (ตัวอย่าง):

```
cd C:\laragon\www\vaelthorn
```
- รันคำสั่งติดตั้งแพ็กเกจ:

```
npm install
```

4) รันโหมดพัฒนา (ให้ดูผลทันที)

```
npm run dev
```

หลังจากรันคำสั่งนี้ คุณจะเห็น URL แบบ `http://localhost:5173` หรือ Laravel dev server (ถ้ามี) — เปิดเว็บเบราว์เซอร์ตาม URL นั้นเพื่อดูหน้า UI.

5) สร้างไฟล์สำหรับนำขึ้นใช้งานจริง (ถ้าต้องการแจกจ่าย)

```
npm run build
```

ปัญหาทั่วไปและวิธีแก้สั้น ๆ
- ข้อความว่า `npm` ไม่รู้จัก: แปลว่ายังไม่ได้ติดตั้ง Node.js — ให้ทำตามขั้นตอนข้อ 2.
- มีข้อผิดพลาดตอน `npm install`: แจ้งข้อความที่ขึ้นมาให้ผม ผมจะช่วยแปลและแนะนำการแก้.

ถ้าคุณต้องการ ผมจะเตรียมคำสั่งเรียกหน้าใน Laravel (Blade) ให้เรียบร้อย หรือรัน `npm install` ให้ แต่ผมไม่สามารถติดตั้งบนเครื่องของคุณจากที่นี่ — ผมจะแนะนำทีละขั้นตอนและช่วยแปลผลลัพธ์ที่คุณส่งกลับมาครับ.

---
คำสั่งที่ผมจะแนะนำให้รันตอนนี้ (คัดลอก-วางใน PowerShell):

```
cd C:\laragon\www\vaelthorn
node -v
npm install
npm run dev
```

ส่งผลลัพธ์ (ข้อความ error หรือ output) กลับมาหากมีปัญหา ผมจะแปลและแก้ให้ทีละขั้นตอนครับ.
